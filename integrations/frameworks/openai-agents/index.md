# OpenAI Agents SDK

> Agent index: [llms.txt](/llms.txt)

AtomicMemory integrates with the OpenAI Agents SDK through an adapter that augments run input with memory and ingests completed turns back into durable storage.

## Deployment targets

### Local

Use this when you want the self-managed OpenAI Agents SDK path with the published adapter package.

See [OpenAI Agents SDK Local](/integrations/frameworks/openai-agents/local).
